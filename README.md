## To run it yourself :)

Please first clone the repo,
- Then create two terminal windows,
- In the first window navigate to the backend folder, and run:
- `npm install`
- `nodemon app.js`
- In the second window navigate to the frontend folder, and run:
- `npm install`
- `npm run dev`
